# Clinic system
An application for managing a medical clinic, including a database for the patient, doctors, appointment schedules, medical records and prescriptions.

## Stack
### Backend:
    - Nest.js
    - Postgres
    - JWT
    - Swagger
### Frontend
    - React.js
    - Next.js
    - Rect-bootstrap

## Running
Go to the project directory, and use next commands to run app:

### Backend:
```bash
# go to the server directory
$ cd server

# install dependencies
$ npm install

# execute app
$ npm run start:dev
```

### Frontend:
```bash
# go to the client directory
$ cd client

# install dependencies
$ npm install

# execute app
$ npm start dev
```
